# moyu-games

[中文](../README.md) | English

moyu-games is a lightweight collection of daily mini-games for quick puzzle breaks.

## Games

- Geography shape guessing
- Sudoku
- Maze
- Lantern riddles
- Idiom guessing
- English practice

## Local Preview

```bash
python3 -m http.server 4177
```

Open:

```text
http://127.0.0.1:4177
```

## Deploy

This is a static site and can be deployed directly to Vercel, GitHub Pages, Cloudflare Pages, Netlify, or any static hosting service.
